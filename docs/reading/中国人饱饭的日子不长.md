# 中国人饱饭的日子不长

阅读推荐，[原文](https://www.163.com/dy/article/JTPL4V9K0556BUK2.html)

>一个国家能有几十年的稳定和富足，靠的不是口号，不是文化基因，也不是某位伟人的画像，而是制度、规则、理性，以及对历史的敬畏。一旦我们失去了这种敬畏，一旦我们习惯了不感恩、不反思、不检讨，我们就会变得浮夸、傲慢、自恋，甚至反过来，憎恨那些曾经拯救我们的原则。那时下一个时代便不再是延续，而是断裂。我们会再次看到规则崩溃、理性消失、极端抬头，我们会再度听见：吃肉是资产阶级腐化的表现，自由是西方的陷阱，法治是为了限制人民斗争的工具。然后你会惊讶地发现，所有你以为早已过去的苦难，都会换上新的面孔，重新出现。

中国有文字记载的三千年历史，能够吃饱饭的年代屈指可数，能吃上肉的年代更是凤毛麟角。至于不打仗、不逃荒、百姓相安的时代，几乎从未存在过。唯一一次出现，正是我们正在经历的这40年。

自加入WTO之后，才第一次让中国人普遍性地吃上肉。请注意是普遍性，不是王公贵族，也不是沿海富人，而是无数个农村家庭、小城镇的夫妻、城市的打工者，都开始有肉吃，有点余钱可以做点小生意，可以在街头巷尾喝酒吃串、养猫逗狗。

当然，改革开放不是自来水龙头拧开就有的，是历史给我们开的一扇罕见的绿灯，是拿掉意识形态的紧箍咒，吞下市场化的苦药水，承受全球化的剧痛换来的。是因为我们不再闭目塞听，不再画地为牢，不再对世界喊打喊杀，才得以第一次以一个普通国家的姿态，站在了全球规则的流水线上。

中国历史不是没有富裕过，盛唐、康乾、宋元交替之间，总有那么些高光时刻，但从未像现在这样富而广、安而久。你以为这是正常？不，这叫奇迹。正因为它太特殊了，我们才不能当它是常态。它是一种罕见的政治理性，是对灾难深刻反思后的修正，是无数人用沉默、牺牲与忍耐换来的喘息窗口。而窗口是会关闭的，一旦我们重新走回封闭、自恋、虚假，一旦我们开始敌视市场、压抑言论、漠视法治，一旦我们认为吃不上肉也没关系，吃草也能强大，那么这场来之不易的奇迹，将如晨雾般散去。

所以请记住，我们正在经历的是一个例外，而不是常态。它不是惯性，不是理所当然，是天上掉下来的礼物。认清这一点，是对历史最基本的尊重，也是对未来最起码的负责。

我们真正该思考的，不是我们现在过得还行，而是下一个时代，是否还能赐予我们同样的恩典。历史不是直线，它从不保证今天拥有的，明天还能继续拥有。一个国家能有几十年的稳定和富足，靠的不是口号，不是文化基因，也不是某位伟人的画像，而是制度、规则、理性，以及对历史的敬畏。一旦我们失去了这种敬畏，一旦我们习惯了不感恩、不反思、不检讨，我们就会变得浮夸、傲慢、自恋，甚至反过来，憎恨那些曾经拯救我们的原则。那时下一个时代便不再是延续，而是断裂。我们会再次看到规则崩溃、理性消失、极端抬头，我们会再度听见：吃肉是资产阶级腐化的表现，自由是西方的陷阱，法治是为了限制人民斗争的工具。然后你会惊讶地发现，所有你以为早已过去的苦难，都会换上新的面孔，重新出现。

这不是危言耸听，这是历史最擅长的把戏。我们这一代人，也许还能活在这时代的余晖中，但如果我们不珍惜、不守护、不传承，下一代人将难以理解，什么是有尊严的活着，将感受不到制度应有的温度，更无法体会拥有选择的权利，是一种多么宝贵的幸运。

这个时代已经对我们足够温柔，请不要把它交还给野蛮。

我再饶舌几句，改开以来的40年，我们享受了祖辈400年、甚至4000年都没有享受过的福祉。我的出生地是赣东北一个普通的村庄，远谈不上富裕，但是，有90%的家庭盖上了三层以上的小洋楼，40%以上的家庭都拥有私家车，有的一家两三部。村村通了高等级公路，户户通了自来水，这在改革开放以前是不可想象的。

生活在开元盛世的老百姓，是体会不到安史之乱灾民挨饿的滋味的。但是挨饿曾是中国人的常态。现在咱升斗小民，又面临着一次重大的历史选择，吃饭还是吃草，一念花落，一念天堂，一念地狱。

佛渡有缘人。
