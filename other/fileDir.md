# 文件合并

在 Linux 上可以通过 `rsync` 命令将文件夹 B 合并到文件夹 A，同时保持文件的相对路径不变。对于相同名称的文件，只保留 A 文件夹中的文件，不会覆盖。这是具体的命令：

```bash
rsync -av --ignore-existing B/ A/
```

## 命令详解

- `rsync`：用于同步文件和文件夹的命令。
- `-a`：归档模式，保持文件的属性、权限等。
- `-v`：显示详细的进程信息。
- `--ignore-existing`：忽略已存在于目标文件夹中的文件，避免覆盖 A 文件夹中的同名文件。

## 示例结果

在执行后，文件夹 A 将包含以下文件：

```css
A/x/y/z1
A/q/w/e1
A/x/y/z2
A/q/w/e2
```

这样就达到了合并文件夹并避免同名文件覆盖的目的。
